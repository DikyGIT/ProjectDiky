TENTANG KAMI
Smart Learning adalah brand identity dari PT Tiga Samudera Teknologi untuk pengembangan perangkat lunak di bidang pendidikan. Smart Multimedia mampu mendorong minat siswa untuk belajar secara mandiri di rumah dan mampu meningkatkan prestasi siswa secara signifikan dalam pencapaian siswa menuju ke jenjang pendidikan Perguruan Tinggi khususnya Perguruan Tingggi Negeri. Smart CBT merupakan perangkat lunak lanjutan dengan mengefektifkan dan mengefisiensikan sumber daya yang ada untuk mendampingi sekolah menuju pengembangan dan peningkatan mutu pendidikannya.
